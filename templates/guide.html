This page is an informal guide on how this api can be used.

This API has 2 endpoints


https://taxy-298609.ts.r.appspot.com/get_trans

GET Request Only

Returns a transaction's information given the transaction_id
example: https://taxy-298609.ts.r.appspot.com/get_trans?transaction_id=0117656a0ab05q0V

https://taxy-298609.ts.r.appspot.com/get_trans_in_date

GET Request Only

Returns all transactions' information in a given date range.

Date format is YYYY-MM-DD.

example:https://taxy-298609.ts.r.appspot.com/get_trans_in_date?start=2019-12-1&end=2021-12-20

alternatively, omit the dates to return all data

example:https://taxy-298609.ts.r.appspot.com/get_trans_in_date

https://taxy-298609.ts.r.appspot.com/update

POST Request Only

Allows updating a transaction record
Payload must be json and must contain a transaction_id field.
